Parcel 2 + React 17 + TypeScript 4 made easy.

## Getting Started

```bash
yarn

# start application
yarn start

# some extra options
yarn start --lazy --https --log-level=verbose

# build for elder browsers
yarn build:legacy

# build for modern browsers
yarn build:modern

# watch build for modern browsers
yarn build:watch

# execute build:legacy and build:modern
yarn build
```
